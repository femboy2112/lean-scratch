## Internal Helper / Compatibility Only

These declarations are accounted for above but should not be cited as new theory primitives:

- Witness constants and finite examples in `TLICA.NonReducibility` are proof witnesses for FND-012.
- Namespace projection theorems such as `FreeWillWitness.hasLiveAlternatives`, `AgencyWitness.selected_is_feasible`, and `PCEFreeWillWitness.toFreeWillWitness` are bridge/accessor results under their parent Rosetta IDs.
- `GeneralProjectMap`, `GeneralProjectedPCE`, and all `general*` names are compatibility-only. The primary projection API is `ProjectMap α Act` with `ProjectedPCE`, and active application modules use the direct API where proof-stable.
- Deferred markers ending in `_deferred` are not theorems of the deferred concept; they are explicit placeholders.
